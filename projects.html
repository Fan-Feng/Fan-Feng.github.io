  • Large Scale Simulation Helps States and Utilities Enhancing Building Energy Code Adoption

    2025

    "I am one of the main contributors to the Residential Building Energy Code analysis programs, where I helped develop the simulation infrastructure used for large-scale building energy analysis at both the state and national levels. Specficially, the simulation program we used are mainly EnergyPlus, and we developed a workflow that can atumatically generate simulation models for different modeling parameters(building types, insulation levels ),climate zones, and etc. The large scale simulation are then executed using High Performance Computers. "

  • Cost-effective Thermally Activated Building Systems with PCM embeded radiant cooling systems to Support a Power Grid System

    2022

    (1) Design a multi-functional building envelope with both phase change materials (PCM) and a radiant heat/cooling system. (2) Develop and implement a simulation module for industrial-grade PCMs in EnergyPlus. (3) Develop control-oriented models for energy and thermal estimation of buildings using this multi-functional panel. (4) Develop a model-predictive control framework for this system to optimize cost and thermal comfort. (5) Implement the proposed control framework using high-performance computers.
